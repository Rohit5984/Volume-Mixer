# VolMix 🔊

**VolMix** is a lightweight Windows system tray utility that provides quick and precise control over your system's audio. It displays the current master volume percentage and lets you adjust both the master volume and individual application volumes in real time.
Designed to be fast, responsive, and low on system resources, VolMix focuses on one thing—making Windows audio control simple and efficient.

---

## ✨ Features

- 🔊 Displays the current **master volume percentage** in the system tray.
- 🎚️ Real-time **master volume slider** with two-way synchronization.
- 🎵 Control the volume of **individual applications** with dedicated sliders.
- 🔇 Mute or unmute specific applications independently.
- 📍 Popup window opens next to the system tray on left-click and automatically hides when it loses focus.
- 🚀 **Single-instance protection** prevents multiple copies of VolMix from running.
- ⚙️ Optional **Start with Windows** setting available during installation or from the application settings.
- 🗑️ Clean uninstaller removes application files, shortcuts, and registry entries.
